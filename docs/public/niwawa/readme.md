# 凹语言吉祥物——鲵凹凹

![WA 酱三视图](./niwawa-20250613-01.jpg)

鲵凹凹是凹语言的第一个吉祥物，诞生时间大约在2022年的11月。鲵凹凹名字是从凹的谐音——娃娃鱼——而来，而娃娃鱼的学名大鲵。因为大鲵会发出婴儿般的啼哭声，因此得名“娃娃鱼”。
