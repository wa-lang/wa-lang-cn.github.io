# 中文汉字编程语言与国产芯片会师！——凹语言原生支持龙芯指令集

- 时间：2026-02-24
- 撰稿：凹语言开发组
- 转载请注明原文链接：[https://wa-lang.org/smalltalk/st0100.html](https://wa-lang.org/smalltalk/st0100.html)

---

![](/st0100-01.jpg)

近日，凹语言碎碎念文章达到100篇（[凹语言“碎碎念”百帖记](https://wa-lang.org/smalltalk/st0099.html)），现在碎碎念文章编号正式增长到st0100，标志着凹语言开源项目进入新的发展阶段。

同时，凹语言完成龙芯LoongArch架构原生适配，在龙芯3A6000真机实现全流程验证，凹语言汉字编程技术与国产自主芯片底层协同终于迈出关键一步，为国产根软件和根硬件生态深度融合探索新开辟了新的方向。

![](/st0100-02.png)

凹语言通过自研LoongArch专属汉字汇编器汇编器，结合内置的汉字汇编语言编写的运行时包，直接生成架构原生无依赖的ELF可执行文件，无第三方编译框架依赖，同时支持指令、寄存器汉字命名，让汉字编程能直接对接国产芯片底层，在技术层面实现了语言与芯片的深度衔接。

![](/st0100-03.png)

凹语言开发组还开发了配套的辅助工具，比如objdump工具可以将ELF可执行文件以汉字指令集的方式反汇编显示，极大提高了汉字汇编器的调试效率。凹语言不仅实现外部依赖、多平台、全链路工具链自研落地，更为凹语言第二个五年计划的稳健长远发展提供了路线保障，是打通“汉字编程语言-汉字汇编语言-汉字指令集-国产芯片”的自主技术闭环的关键性一步。

![](/st0100-04.jpg)

我们期待以此次适配为起点，后续在性能优化、生态工具完善、工业场景适配等方面进行更深入的探索。未来，凹语言将持续深化与龙芯生态的协同，为国产基础软件与核心硬件的协同创新积累实践经验，通过凹语言英文前端语法结合中文汉字编程语法，在通用编程语言国际化和本土化方向如何融合共存方向进行更多探索。
