# 感谢龙芯对凹语言的支持, 已点亮真机

- 时间：2025-12-18
- 撰稿：凹语言开发组
- 转载请注明原文链接：[https://wa-lang.org/smalltalk/st0097.html](https://wa-lang.org/smalltalk/st0097.html)

---

龙芯 (LoongArch) 作为我国自主设计和发展的高性能芯片，已经建立起相对独立的生态。为了支持国产开源软件的发展，龙芯第一时间给凹语言社区免费提供了最新的3A6000的迷你主机。并且贴心地位我们预先安装了必要的开发软件。主机和开机效果如下：

![](/st0097-01.jpg)

![](/st0097-02.jpg)

凹语言安装好之后，基于WebAssembly后端的编译器已经可以正常工作：

![](/st0097-03.jpg)

为了更好地挖掘性能潜力和适配凹语言前端，凹语言开发者给龙芯CPU从头设计了支持中英文语法汇编器，用于面向本地的编译器开发。目前凹语言的龙芯汇编器已经在真机跑通中文版的例子：

![](/st0097-04.png)

从最底层开始重建生态从来都不是一件容易的事情，感谢龙芯为我们建立了自己的CPU和指令集架构。这让凹语言可以在巨人的肩膀之上，开发符合本土特殊的中文版本的汇编语言和汇编器，从而为全中文编程生态建立提供了一个完美路径。

![](/st0097-05.jpg)

一切刚刚开始，我们共同努力建设本土生态！
