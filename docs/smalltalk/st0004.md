::: tip
`mac` 用户需要 `vscode/settings.json` 中配置 `rootPath`，即 wa 的程序地址
```json
{
  "wa.rootPath": "/Users/xxx/go/bin/wa"
}
```
:::

| 定义结构 | `tys` \| `结构` \| `jiegou`\|`jg` | type name struct {}<br />类型 name 结构 {} |
